# tree

List contents of directories in a tree-like format.

By default, the `tree` may not be installed. Install it with your package manager.

```shell
# Ubuntu
sudo apt install tree
```

## Usage

```shell
tree my-python-app
```

```
my-python-app/
├── pyproject.toml
├── README.md
├── src
│    └── my-python-app
│        └── __init__.py
└── tests
    └── __init__.py
```
