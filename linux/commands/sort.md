# sort

Sort the lines of a fine alpha-numerically and print the sorted result to standard
output.

```shell
sort pets.txt
```

Will return something like

```
dog
dog
dog
humster
humster
parrot
```

## Options

### -r

Get lines sorted in reverse order.

```shell
sort -r pets.txt
```

```
parrot
humster
humster
dog
dog
dog
```
